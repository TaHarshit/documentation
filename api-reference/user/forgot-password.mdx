## Endpoint

```
POST /api/v1/user/forgot_password
```



## Headers

<ParamField header="X-API-Token" type="string" required>
  Your API access token. You can also set this globally in the API playground.
</ParamField>

<ParamField header="platform" type="string" required>
  The platform making the request: `WEB`, `IOS`, or `ANDROID`.
</ParamField>
## Request Body

<ParamField body="email" type="string" required>
  Registered email address to receive reset link.
</ParamField>

## Response

<ResponseField name="message" type="string">
  Success message indicating email was sent
</ResponseField>

## Password Reset Flow

<Steps>
  <Step title="Request Reset">
    User submits their email address to this endpoint
  </Step>
  <Step title="Receive Email">
    User receives an email with a password reset link containing a token
  </Step>
  <Step title="Click Link">
    User clicks the link which directs to `/user/reset_password/{token}`
  </Step>
  <Step title="Set New Password">
    User submits new password via `/user/update_password` endpoint
  </Step>
</Steps>

## Notes

- Rate limited to 3 requests per minute per IP address
- Reset token expires after 60 minutes
- Email must be registered in the system
- Check spam folder if email is not received
- Only one active reset token per user at a time
