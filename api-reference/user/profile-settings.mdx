# Profile Settings

Get your profile settings and preferences.

## Endpoint

```
POST /api/v1/user/profile_settings
```

<Note>This endpoint requires authentication</Note>



## Headers

<ParamField header="X-API-Token" type="string" required>
  Your API access token. You can also set this globally in the API playground.
</ParamField>

<ParamField header="platform" type="string" required>
  The platform making the request: `WEB`, `IOS`, or `ANDROID`.
</ParamField>
## Request Body

```json
{}
```

## Response

<ResponseField name="status" type="string">
  Success status
</ResponseField>

<ResponseField name="data" type="object">
  Response data
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST http://localhost/fss/public/api/v1/user/profile_settings \
  -H "X-API-Token: YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{}'
```

```javascript JavaScript
const response = await fetch('http://localhost/fss/public/api/v1/user/profile_settings', {
  method: 'POST',
  headers: {
    'X-API-Token': 'YOUR_API_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({})
});

const data = await response.json();
```

</RequestExample>
