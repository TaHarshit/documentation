title: 'Sign In'
api: 'POST /api/v1/user/signin'

---

## Endpoint

```
POST /api/v1/user/signin
```



## Headers

<ParamField header="X-API-Token" type="string" required>
  Your API access token. You can also set this globally in the API playground.
</ParamField>

<ParamField header="platform" type="string" required>
  The platform making the request: `WEB`, `IOS`, or `ANDROID`.
</ParamField>
## Request Body

<ParamField body="email" type="string" required>
  Registered email address.
</ParamField>

<ParamField body="password" type="string" required>
  User account password.
</ParamField>

<ParamField body="device_token" type="string">
  Device token for push notifications
</ParamField>

<ParamField body="platform" type="string" required>
  Platform identifier: IOS, ANDROID, or WEB
</ParamField>

## Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Authentication Data">
    <ResponseField name="api_token" type="string">
      Bearer token for authentication
    </ResponseField>
    <ResponseField name="user" type="object">
      Complete user profile information
    </ResponseField>
  </Expandable>
</ResponseField>

const response = await fetch('http://fssportal.com/api/v1/user/signin', {
  method: 'POST',

```python Python
import requests

response = requests.post(
    'http://fssportal.com/api/v1/user/signin',
    headers={
        'X-API-Token': 'YOUR_API_TOKEN',
        'Content-Type': 'application/json'
    },
    json={
        'email': 'john.doe@example.com',
        'password': 'SecurePass123',
        'device_token': '',
        'platform': 'WEB'
    }
)

data = response.json()
token = data['data']['api_token']
```

```php PHP
<?php

$curl = curl_init();

curl_setopt_array($curl, [
  CURLOPT_URL => 'http://fssportal.com/api/v1/user/signin',
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_POST => true,
  CURLOPT_HTTPHEADER => [
    'X-API-Token: YOUR_API_TOKEN',
    'Content-Type: application/json'
  ],
  CURLOPT_POSTFIELDS => json_encode([

    'password' => 'SecurePass123',
    'device_token' => '',
    'platform' => 'WEB'
  ])
]);

$response = curl_exec($curl);
curl_close($curl);

$data = json_decode($response, true);
$token = $data['data']['api_token'];
```

## Notes

- Store the `api_token` securely for subsequent API requests
- Include the token in the Authorization header: `Bearer {api_token}`
- Device token is optional but recommended for push notifications
- Failed login attempts may trigger rate limiting
- Account must be active (status = 1) to sign in
