# Invite User

Invite a new user to join your company.

## Endpoint

```
POST /api/v1/user/invite_user
```

<Note>This endpoint requires authentication</Note>



## Headers

<ParamField header="X-API-Token" type="string" required>
  Your API access token. You can also set this globally in the API playground.
</ParamField>

<ParamField header="platform" type="string" required>
  The platform making the request: `WEB`, `IOS`, or `ANDROID`.
</ParamField>

<ParamField header="Authorization" type="string" required placeholder="Bearer {token}">
  User authentication token (Bearer token received after signin).
</ParamField>
## Request Body

<ParamField body="email" type="string">
  Email of user to invite
</ParamField>

<ParamField body="role" type="string">
  User role (admin, member)
</ParamField>

## Response

<ResponseField name="status" type="string">
  Success status
</ResponseField>

<ResponseField name="data" type="object">
  Response data
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST http://fssportal.com/api/v1/user/invite_user \
  -H "X-API-Token: YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "newuser@example.com",
    "role": "member"
  }'
```

```javascript JavaScript
const response = await fetch('http://fssportal.com/api/v1/user/invite_user', {
  method: 'POST',
  headers: {
    'X-API-Token': 'YOUR_API_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    email: 'newuser@example.com',
    role: 'member'
  })
});

const data = await response.json();
```

</RequestExample>
