---

## Endpoint

```
GET /api/v1/user/reset_password/{token}
```

## Path Parameters

<ParamField path="token" type="string" required>
  Password reset token received via email
</ParamField>



## Headers

<ParamField header="X-API-Token" type="string" required>
  Your API access token. You can also set this globally in the API playground.
</ParamField>

<ParamField header="platform" type="string" required>
  The platform making the request: `WEB`, `IOS`, or `ANDROID`.
</ParamField>
## Response

<ResponseField name="message" type="string">
  Token validation message
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Token Data">
    <ResponseField name="token" type="string">
      Validated reset token
    </ResponseField>
    <ResponseField name="email" type="string">
      Email associated with the token
    </ResponseField>
  </Expandable>
</ResponseField>

```

{

  "status": 410

```

</ResponseExample>

## Notes

- Token is valid for 60 minutes from generation
- Token can only be used once
- After validation, use `/user/update_password` to set new password
- Invalid tokens should redirect user to request a new reset
