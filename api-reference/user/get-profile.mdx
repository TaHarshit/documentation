## Endpoint

```
POST /api/v1/user/get_profile
```

<Note>This endpoint requires authentication</Note>



## Headers

<ParamField header="X-API-Token" type="string" required>
  Your API access token. You can also set this globally in the API playground.
</ParamField>

<ParamField header="platform" type="string" required>
  The platform making the request: `WEB`, `IOS`, or `ANDROID`.
</ParamField>
## Request Body

No request body required.

## Notes

- If `user_id` is not provided, returns the authenticated user's profile
- Requires active subscription
- Profile image URL may be empty if no image uploaded
- `field_display_orders` contains custom field ordering preferences
- Company owners have additional permissions
