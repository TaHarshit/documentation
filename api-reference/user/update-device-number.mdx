# Update Device Number

Update your registered device number.

## Endpoint

```
POST /api/v1/user/update_devicenumber
```

<Note>This endpoint requires authentication</Note>



## Headers

<ParamField header="X-API-Token" type="string" required>
  Your API access token. You can also set this globally in the API playground.
</ParamField>

<ParamField header="platform" type="string" required>
  The platform making the request: `WEB`, `IOS`, or `ANDROID`.
</ParamField>

<ParamField header="Authorization" type="string" required placeholder="Bearer {token}">
  User authentication token (Bearer token received after signin).
</ParamField>
## Request Body

<ParamField body="device_number" type="string">
  Device number
</ParamField>

## Response

<ResponseField name="status" type="string">
  Success status
</ResponseField>

<ResponseField name="data" type="object">
  Response data
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST http://localhost/fss/public/api/v1/user/update_devicenumber \
  -H "X-API-Token: YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "device_number": "1234567890"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost/fss/public/api/v1/user/update_devicenumber', {
  method: 'POST',
  headers: {
    'X-API-Token': 'YOUR_API_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    device_number: '1234567890'
  })
});

const data = await response.json();
```

</RequestExample>
