# Update Profile

Update your user profile information.

## Endpoint

```
POST /api/v1/user/update_profile
```

<Note>This endpoint requires authentication</Note>



## Headers

<ParamField header="X-API-Token" type="string" required>
  Your API access token. You can also set this globally in the API playground.
</ParamField>

<ParamField header="platform" type="string" required>
  The platform making the request: `WEB`, `IOS`, or `ANDROID`.
</ParamField>

<ParamField header="Authorization" type="string" required placeholder="Bearer {token}">
  User authentication token (Bearer token received after signin).
</ParamField>
## Request Body

<ParamField name="first_name" type="string" required={false}>
  First name
</ParamField>

<ParamField name="last_name" type="string" required={false}>
  Last name
</ParamField>

<ParamField name="phone" type="string" required={false}>
  Phone number
</ParamField>

<ParamField name="email" type="string" required={false}>
  Email address
</ParamField>

## Response

<ResponseField name="status" type="string">
  Success status
</ResponseField>

<ResponseField name="data" type="object">
  Response data
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST http://localhost/fss/public/api/v1/user/update_profile \
  -H "X-API-Token: YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "first_name": "John",
    "last_name": "Doe",
    "phone": "1234567890"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost/fss/public/api/v1/user/update_profile', {
  method: 'POST',
  headers: {
    'X-API-Token': 'YOUR_API_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    first_name: 'John',
    last_name: 'Doe',
    phone: '1234567890'
  })
});

const data = await response.json();
```

</RequestExample>
