# Delete Measurement Image

Delete a measurement photo.

## Endpoint

```
POST /api/v1/measurements/delete-image
```

<Note>This endpoint requires authentication</Note>



## Headers

<ParamField header="X-API-Token" type="string" required>
  Your API access token. You can also set this globally in the API playground.
</ParamField>

<ParamField header="platform" type="string" required>
  The platform making the request: `WEB`, `IOS`, or `ANDROID`.
</ParamField>
## Request Body

<ParamField body="image_id" type="integer">
  Image ID
</ParamField>

## Response

<ResponseField name="status" type="string">
  Success status
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST http://localhost/fss/public/api/v1/measurements/delete-image \
  -H "X-API-Token: YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "image_id": 456
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost/fss/public/api/v1/measurements/delete-image', {
  method: 'POST',
  headers: {
    'X-API-Token': 'YOUR_API_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    image_id: 456
  })
});

const data = await response.json();
```

</RequestExample>
