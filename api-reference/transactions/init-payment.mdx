# Init Payment

Initialize a subscription payment.

## Endpoint

```
POST /api/v1/transactions/init-payment
```

<Note>This endpoint requires authentication</Note>



## Headers

<ParamField header="X-API-Token" type="string" required>
  Your API access token. You can also set this globally in the API playground.
</ParamField>

<ParamField header="platform" type="string" required>
  The platform making the request: `WEB`, `IOS`, or `ANDROID`.
</ParamField>

<ParamField header="Authorization" type="string" required placeholder="Bearer {token}">
  User authentication token (Bearer token received after signin).
</ParamField>
## Request Body

<ParamField body="plan_id" type="integer">
  Plan ID
</ParamField>

<ParamField body="payment_method" type="string">
  Payment method (card, etc.)
</ParamField>

## Response

<ResponseField name="status" type="string">
  Success status
</ResponseField>

<ResponseField name="data" type="object">
  Payment intent data
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST http://localhost/fss/public/api/v1/transactions/init-payment \
  -H "X-API-Token: YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "plan_id": 1,
    "payment_method": "card"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost/fss/public/api/v1/transactions/init-payment', {
  method: 'POST',
  headers: {
    'X-API-Token': 'YOUR_API_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    plan_id: 1,
    payment_method: 'card'
  })
});

const data = await response.json();
```

</RequestExample>
